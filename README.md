# Тестовое задание 
 
Чтобы запустить проект нужно запустить клиент и сервер по отдельности.

## Запуск клиента

### cd client

### npm install

### npm run serve

## Запуск сервера

### cd laravel

### php artisan serve

Все миграции уже сделаны. База храниться локально на компьютере. 

### Использованный стэк: Vue.js, Laravel, MySQL
